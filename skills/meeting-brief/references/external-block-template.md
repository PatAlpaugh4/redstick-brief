# External Meeting Block — Template

Use this exact format for every meeting where at least one attendee is NOT `@redstickvc.com`.

```
<TIME> · <FIRST_NAME> · <COMPANY_NAME>
<MEETING_TYPE> · <CONTEXT_LINE>

ORIENT
<2–4 lines on who they are and current state. Include scorecard verdict + confidence if a scorecard exists for this company. Cite specifics.>

WHERE YOU LEFT OFF
<2–4 lines on prior conversation specifics — what was promised, what's outstanding, what was the last beat. Cite emails by date.>

WHAT'S NEW (last 14 days)
<2–5 bullets on recent public activity, news, team changes, anomalies. Each bullet cited with URL/source + date. Cut bullets older than 14 days unless decision-relevant.>

FIRST MOVE
"<Literal opening line Cam can speak in the meeting.>"
<Optional one-line rationale in parentheses.>

THE ANGLE
<1–3 sentences on what to prioritize in the meeting and why. Pick a read; no hedging.>

ONE THING
<Single memorable insight unique to this meeting. The "wow" beat. Cut if not genuinely sharp.>
```

## Section rules

- **<MEETING_TYPE>** is one of: `Founder call`, `Founder follow-up`, `Portfolio check-in`, `LP intro`, `LP update`, `Co-investor sync`, `Inbound intro`. Pick the closest fit from the event title and Gmail context.
- **<CONTEXT_LINE>** is a one-liner: `second touch since Tuesday's intro`, `first call`, `monthly update`, `intro from Marcus`, `recurring 1:1`. Cut it if there's no real context.
- **ORIENT** anchors Cam: who, what they do, what stage, what's the current state. If a scorecard exists, surface the verdict + confidence in one phrase. Do NOT re-state the deck — pull the sharpest 2–4 lines.
- **WHERE YOU LEFT OFF** is the highest-leverage section. Be specific: "She promised a financial model 'this week' (per Tuesday email). Hasn't sent." If this is a first-touch meeting, delete the WHERE YOU LEFT OFF section entirely.
- **WHAT'S NEW** uses the last-14-day window. Each bullet cites a source. If nothing real surfaced, delete the section. Do not pad.
- **FIRST MOVE** is the literal opening line, in quotes. Not "ask about X" — the actual sentence Cam will speak. The optional rationale is one short line.
- **THE ANGLE** picks priorities. "Bowery pilot first, engineer transition second, unit econ third." No hedging.
- **ONE THING** is the bonus insight — only include if it's genuinely sharp. Otherwise cut.

## Citation format

Inline, italic, parenthetical:

- `*(per Tuesday email)*`
- `*(LinkedIn yesterday)*`
- `*(AgFunder 4/29)*`
- `*(scorecard 4/30)*`
- `*(her tweet 5/3)*`

Always include source + relative or absolute date.

## Length target

8–18 lines total per block. A 4-line block on a thin meeting beats a 12-line block of filler.
